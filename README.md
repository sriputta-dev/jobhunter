# JobHunter Agent 🤖

A multi-agent AI system that automates the most painful parts of job searching. Upload your resume, paste any job URL or description from LinkedIn, Indeed, Dice, or any company careers page, and get back an ATS fit score, tailored resume bullets, a personalized cold email, and interview strategy - all generated by four specialized AI agents working in sequence, powered by Claude.

## What You Actually Get

**Immediately, no API key needed:**

An ATS fit score from 0 to 100 showing how well a job matches your skill profile, with the exact keywords that matched and which ones are missing.

**After clicking Run AI Analysis (needs Anthropic key, ~$5 free credits):**

Four agents run in sequence, each building on the previous output.

The Job Analyst reads your resume alongside the job description and returns a structured fit assessment - STRONG FIT, GOOD FIT, STRETCH, or SKIP - with clear reasoning. The Resume Tailor rewrites your summary and bullet points using the exact language of the JD, grounded in your actual resume content. The Outreach Specialist drafts a personalized cold email under 120 words with a specific hook for that company. The Strategy Coach generates five technical interview questions with hints drawn from your real projects, behavioral questions with STAR method outlines, a company research tip, and your unique angle for that role.

Everything is editable. Click Edit on any AI output, make your changes, save, and revert to the original anytime. Both versions are stored.

## How Jobs Get Added

**Paste Mode** is the main way to use this. Find a job on LinkedIn, Indeed, Dice, Glassdoor, Greenhouse, Lever, or any company site. Copy either the URL or the job description text. Paste it into the Paste Job tab. The system auto-detects which one it received, extracts the content, scores it, and adds it to your tracker in under two seconds.

If a URL does not load because a site blocks automated fetching, just copy and paste the job description text directly. Text mode works everywhere.

**Auto Search** pulls from five free job boards simultaneously - Himalayas, Remotive, Jobicy, Arbeitnow, and Indeed RSS - with results scored and ready to review.

## Setup

**1. Clone**

```bash
git clone https://github.com/your-username/jobhunter.git
cd jobhunter
```

**2. Configure**

```bash
cp backend/.env.example backend/.env
```

Open `backend/.env` and add your key:

```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

To get your key: go to console.anthropic.com, sign up free, go to API Keys, click Create Key. New accounts get $5 in credits which covers around 100 job analyses.

**3. Backend**

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn api.main:app --reload --port 8000
```

Windows: use `venv\Scripts\activate` instead. You should see `Application startup complete`.

**4. Frontend**

Open a new terminal:

```bash
cd frontend
npm install
npm start
```

Dashboard opens at http://localhost:3000.

## First Thing To Do - Upload Your Resume

Before running AI analysis, upload your resume so agents use your actual experience rather than the sample placeholder.

Click the resume area at the top of the left panel. Select your `.pdf` or `.docx` file. Text is extracted immediately and agents use it from that point forward, including after server restarts.

If your PDF is a scanned image rather than text-based, upload the `.docx` version instead. The system will tell you clearly if extraction fails and what to do.

## Customize for Your Profile

Open `backend/tools/ats_scorer.py` and edit the `CANDIDATE_PROFILE` dictionary. The primary skills list drives 70 percent of the ATS score so accuracy here matters most. Replace the default .NET/React skill set with your actual stack.

Open `backend/agents/agents.py` and edit `DEFAULT_CANDIDATE_BACKGROUND`. This is what agents fall back to when no resume is uploaded. Once you upload a resume file it overrides this automatically.

## Using the Dashboard

Paste a job by clicking the Paste Job tab, entering a URL or description text, and clicking Add Job. The job appears with an ATS score in under two seconds.

Search for jobs by clicking Search Jobs, entering a title and location, and clicking Search. Results from five boards appear automatically.

Run AI analysis by clicking any job in the list, then clicking Run AI Analysis. Results come back in 30 to 90 seconds depending on API speed. Click the refresh button after waiting.

Edit AI output by clicking Edit on the Resume, Cold Email, or Strategy tabs. A textarea opens with the content. Save your version and click Revert to AI anytime to restore the original.

Track applications using the Status dropdown on any job - New, Applied, Interviewing, Rejected, Offered.

## Adding a New Job Source

Open `backend/tools/job_scraper.py` and add a function:

```python
def scrape_myboard(query: str, limit: int = 10) -> List[Dict]:
    jobs = []
    resp = requests.get(f"https://myboard.com/api/jobs?q={query}", timeout=15)
    for job in resp.json().get("results", [])[:limit]:
        jobs.append({
            "title": job["title"],
            "company": job["company"],
            "location": job.get("location", "Remote"),
            "url": job["url"],
            "description": job["description"],
            "salary": job.get("salary", ""),
            "source": "myboard",
            "posted_date": job.get("date", ""),
        })
    return jobs
```

Then add it to the `SOURCES` list inside `fetch_all_jobs()`. Deduplication and scoring happen automatically.

## API Reference

| Endpoint | Method | Description |
|---|---|---|
| GET /health | GET | Health check |
| GET /api/jobs | GET | List jobs with optional status, min_score, source filters |

| GET /api/resume | GET | Get current resume profile and preview |
| DELETE /api/resume | DELETE | Remove uploaded resume |
| GET /api/stats | GET | Dashboard statistics |

## Troubleshooting

Backend does not start: make sure the virtual environment is active (`source venv/bin/activate`) and dependencies are installed (`pip install -r requirements.txt`).

Dashboard shows "Failed to load jobs": confirm the backend is running. Test with `curl http://localhost:8000/health` - should return `{"status":"ok"}`.

URL paste returns nothing: some sites block automated fetching. Copy and paste the job description text directly instead. This works everywhere.

AI analysis not appearing: check `ANTHROPIC_API_KEY` in `backend/.env`. Check the backend terminal for error messages.

ATS scores look off: edit `CANDIDATE_PROFILE` in `backend/tools/ats_scorer.py` with your real skills.

Resume not affecting AI output: make sure your PDF is text-based not a scanned image. Try uploading the DOCX version.
